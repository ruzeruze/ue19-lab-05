# ue19-lab-05
Ce repo permet de créer un conteneur qui utilise l'API chucknorris.io pour printer des blagues à propos de Chuck Norris. Un script python fait une requête vers api.chucknnorris.io/jokes/random afin de recevoir une blague de l'API, puis l'affiche à l'aide d'un print().
Afin d'utiliser ce projet, il faut construire l'image du conteneur, puis l'executer à l'aide de Docker.
